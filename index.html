    Help

    How Stride works

    A calm map of the workspace, so you can use the parts you need and ignore the rest.

    Sidebar and direction controls

    The sidebar keeps each workspace section one tap away. Use the back and forward buttons to step through the flow in order, from overview to reset tools.

    Tasks and progress

    Add specific next actions, give them a priority, filter the list by status, and let the Top 3 strip keep the most useful work visible.

    Useful links and options

    Quick Launch opens the outside tools you probably use anyway, while Focus mode hides the playful extras so you can work with less visual drag.

    Game, notes, and exports

    The arcade now has five quick games, reflection is for short learning notes, and the file window is for longer working drafts. Everything here saves locally.
